

##  Funcionalidades

### Página Inicial

-   Apresentação da plataforma\
-   Botão de acesso rápido para a calculadora de consumo\
-   Destaque para os principais benefícios:\
    ✔ Transparência no consumo\
    ✔ Economia financeira\
    ✔ Sustentabilidade\
    ✔ Dicas inteligentes

###  Calculadora de Consumo

Permite que o usuário informe:\
- Potência do equipamento\
- Horas de uso por dia\
- Valor da tarifa\
E receba o cálculo automático do gasto mensal e anual.

###  Sobre

Explica o propósito da plataforma, sua importância e como os cálculos
são realizados.

###  Contato

Canal de comunicação direto para feedbacks e dúvidas.
